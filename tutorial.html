Tutorial

  1. User diharuskan register / daftar terlebih dahulu
  2. Jika sudah memiliki akun, silahkan login
  3. Setelah login selesai, user akan masuk ke halaman Mahasiswa
  4. Pilih menu vote, nanti akan menampilkan daftar calon kandidat
  5. Klik vote
  6. Setelah vote berhasil, silahkan Log out akun user.
